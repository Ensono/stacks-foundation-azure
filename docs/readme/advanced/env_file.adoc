
=== Environment File Generation

[source,hcl]
----
module "foundation" {
  source = "./stacks-foundation-azure"

  company_name       = "Ensono"
  location           = "uksouth"
  project            = ["webapp"]
  stage_name         = "networking"
  environment        = "dev"
  generate_env_files = true
  output_path        = "./generated"
  
  outputs = jsonencode({
    dev = {
      resource_group_name = azurerm_resource_group.main.name
      location            = var.location
    }
  })
}
----

This generates:
- `dev-networking-envvars.bash`
- `dev-networking-envvars.ps1`
- `dev-networking-inputs.auto.tfvars`
